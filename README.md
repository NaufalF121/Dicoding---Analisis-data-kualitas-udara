# Dicoding - Analisis Data Kualitas Udara

## How to run
- Init venv
```
python -m venv venv
```

- Activate venv <br>

Linux
```
source venv/bin/activate
```
Windows
```
venv\Scripts\activate
```

- Installing dependencies
```
pip install -r requirements.txt
```

- Run streamlit
```
streamlit run ./dashboard/dashboard.py
```
